# 기존 전역 지침 보호

## 목적

`install-global-instructions.sh`가 사용자의 기존 글로벌 지침을 검사 없이 덮어쓰지 않도록 한다.

## 작업

- 생성 파일에 스크립트 관리 여부를 판별할 수 있는 명확한 마커를 넣는다.
- 관리되지 않은 대상 파일이 이미 있으면 기본 실행을 중단하고 안내한다.
- 명시적으로 교체할 때는 타임스탬프가 포함된 백업을 먼저 만든다.
- 임시 파일이 오류 발생 후 남지 않도록 `trap`으로 정리한다.
- 완성된 임시 파일만 대상 위치로 이동한다.

## 완료 조건

- 기존 사용자 설정이 기본 실행으로 유실되지 않는다.
- 스크립트가 이전에 생성한 파일은 안전하게 갱신할 수 있다.
- 실패한 실행 뒤에 불필요한 임시 파일이 남지 않는다.
