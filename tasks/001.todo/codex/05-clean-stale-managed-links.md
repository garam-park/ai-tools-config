# 삭제된 스킬의 오래된 링크 정리

## 목적

원본에서 제거된 스킬의 심볼릭 링크가 도구별 대상 경로에 계속 남는 문제를 해결한다.

## 작업

- 스크립트가 생성한 링크 목록을 manifest 등으로 기록한다.
- 이전 manifest에는 있지만 현재 원본에는 없는 스킬을 탐지한다.
- 해당 항목이 여전히 스크립트가 만든 심볼릭 링크일 때만 제거한다.
- 사용자가 링크를 파일이나 디렉터리로 교체한 경우 삭제하지 않고 경고한다.
- 설치 성공 후 manifest를 현재 상태로 갱신한다.

## 완료 조건

- 원본에서 스킬을 삭제한 뒤 설치 스크립트를 실행하면 관리 대상의 오래된 링크만 제거된다.
- 사용자 소유 항목은 제거되지 않는다.
