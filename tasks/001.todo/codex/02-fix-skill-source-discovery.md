# 스킬 원본 경로와 검색 범위 수정

## 목적

저장소 루트에서 `install-skills.sh`를 실행해도 `skills/` 아래의 실제 스킬만 설치되도록 한다.

## 작업

- 스크립트 위치를 저장소 루트로 계산한다.
- 원본 스킬 경로를 `<저장소>/skills`로 고정한다.
- `SKILL.md`가 있는 하위 디렉터리만 스킬로 취급한다.
- 스킬이 하나도 없을 때 glob 문자열을 잘못 처리하지 않고 정상적으로 안내한다.
- `global-instructions/`나 `skills/` 자체가 스킬로 링크되지 않게 한다.

## 완료 조건

- 저장소 루트에서 직접 실행할 수 있다.
- `paced-explainer`만 스킬 대상으로 탐지된다.
- 잘못된 구조의 디렉터리는 건너뛰거나 명확한 경고를 출력한다.
