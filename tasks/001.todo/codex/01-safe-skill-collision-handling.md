# 스킬 설치 시 기존 파일 보호

## 목적

`install-skills.sh`가 대상 경로의 기존 파일이나 디렉터리를 무조건 삭제하지 않도록 수정한다.

## 작업

- 기존 항목이 올바른 심볼릭 링크면 변경하지 않는다.
- 스크립트가 만든 잘못된 심볼릭 링크만 안전하게 교체한다.
- 일반 파일 또는 실제 디렉터리가 충돌하면 삭제하지 말고 오류와 해결 방법을 출력한다.
- 강제 교체가 필요하다면 명시적인 `--force` 옵션에서만 허용하고, 교체 전에 백업한다.

## 완료 조건

- 기본 실행에서 사용자 파일과 디렉터리가 삭제되지 않는다.
- 같은 명령을 반복 실행해도 결과가 동일하다.
- 충돌 상황을 사용자가 이해할 수 있는 메시지로 안내한다.
