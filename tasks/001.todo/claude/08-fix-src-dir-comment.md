## 문제

L14 주석: `# 원본 폴더 (이 스크립트가 있는 곳의 부모)` — "부모(parent)"라고 설명.
L15 코드: `SRC_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"` — 실제로는 **스크립트 자기 폴더**를 반환한다(`..` 없음).
주석이 코드를 잘못 설명해, 스킬 루트가 어디인지 오해를 유발한다. (`install-global-instructions.sh:12`는 같은 관용구를 올바른 주석과 함께 쓴다.)

## 수정 방향

주석을 코드와 일치시킨다:

```bash
# 원본 폴더 (이 스크립트가 있는 폴더)
```
