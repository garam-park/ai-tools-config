# 02. `rm -rf`를 심볼릭 링크에만 적용하도록 가드

- **우선순위**: P1 (데이터 손실 위험)
- **대상 파일**: `install-skills.sh` (L37–L39)
- **상태**: TODO

## 문제

기존 링크를 교체할 때 `rm -rf "$link"`를 심링크 여부 확인 없이 실행한다.
대상 경로에 **실제 디렉토리**(예: 마이그레이션 중인 사용자가 손으로 편집한 `~/.claude/skills/paced-explainer/`, 아직 리포에 없는 로컬 전용 콘텐츠)가 있으면 재귀 삭제되어 **조용히 소실**된다. 주석 자체가 "파일/링크/디렉토리 모두" 제거한다고 인정하고 있다.

## 근거 (현재 코드)

```bash
# 기존 항목 제거 (파일/링크/디렉토리 모두)
rm -rf "$link"          # L38 — 실제 디렉토리도 지움
ln -s "$skill_dir" "$link"
```

## 수정 방향

심링크일 때만 제거하고, 실제 파일/디렉토리는 건드리지 않고 건너뛴다.

```bash
# 심볼릭 링크만 교체한다. 실제 파일/디렉토리는 보존한다.
if [[ -L "$link" ]]; then
  rm -f "$link"
elif [[ -e "$link" ]]; then
  echo "skip: $link 은(는) 실제 파일/디렉토리라 덮어쓰지 않음" >&2
  continue
fi
ln -s "$skill_dir" "$link"
```

## 완료 조건

- [ ] `$link`가 심링크면 정상 교체된다 (멱등성 유지)
- [ ] `$link`가 실제 디렉토리/파일이면 삭제하지 않고 경고 후 건너뛴다
- [ ] 최초 실행(대상 없음)에서도 정상 동작한다
