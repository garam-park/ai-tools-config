# 10. install-skills.sh에 nullglob 설정

- **우선순위**: P3 (사소한 정리 — 셸 견고성)
- **대상 파일**: `install-skills.sh` (L32 앞)
- **상태**: TODO

## 문제

기본값에서 `nullglob`이 꺼져 있어, `"$SRC_DIR"/*/`가 아무것도 매칭하지 못하면
Bash는 패턴을 그대로 남긴다. 그러면 루프가 `skill_dir = "$SRC_DIR"/*/` 리터럴로 한 번 돌고,
`name`이 `*`가 되어 `ln -s`로 이름이 `*`인 깨진 심링크가 생성된다.
`ln`이 성공하므로 `set -euo pipefail`에서도 중단되지 않아 **조용히** 발생한다.

(실제 트리거되려면 스크립트를 하위 디렉토리 없는 폴더에 두어야 해 드문 경우지만, 방어는 한 줄로 끝난다.)

## 수정 방향

루프 앞에 추가:

```bash
shopt -s nullglob
```

이러면 매칭 없을 때 루프가 0회 실행된다.

## 완료 조건

- [ ] 하위 디렉토리가 없는 경우 루프가 0회 돌고 `*` 심링크가 생기지 않는다
- [ ] 정상 케이스 동작은 변하지 않는다
