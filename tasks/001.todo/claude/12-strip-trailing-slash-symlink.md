# 12. 심링크 타깃의 후행 슬래시 제거

- **우선순위**: P3 (사소한 정리 — 미관)
- **대상 파일**: `install-skills.sh` (L39, 루프 본문)
- **상태**: TODO

## 문제

`skill_dir` 값은 `*/` 글롭에서 오므로 항상 `/`로 끝난다.
따라서 `ln -s "$skill_dir" "$link"`가 저장하는 타깃 문자열도 `/`로 끝나
(`paced-explainer -> /abs/paced-explainer/`), `readlink` 결과가 비정규 형태가 된다.
Linux에서 해석은 정상이지만, 심링크 타깃을 문자열 비교하는 도구엔 노이즈다. **동작엔 무해.**

## 수정 방향

루프 본문 상단에서 후행 슬래시를 제거한다.

```bash
skill_dir="${skill_dir%/}"
```

(`name`/`basename` 로직은 이미 슬래시 유무와 무관하게 동작한다.)
