# 공통 지침

## 커뮤니케이션

- 한국어로 대화해요.
- 결과와 중요한 판단을 먼저 간결하게 설명해요.
- 확인할 수 있는 내용은 먼저 직접 확인해요.
- 결과를 크게 바꾸는 결정이 필요할 때만 질문해요.

## 작업 원칙

- 사용자가 요청한 범위 안에서만 작업해요.
- 기존 사용자 변경과 관련 없는 파일은 건드리지 않아요.
- 변경 전에는 무엇을 바꿀지 짧게 알려요.
- 변경 후에는 위험도에 맞는 검증을 수행하고 결과를 알려요.

## Git

- 사용자가 명시적으로 요청한 경우에만 커밋하거나 push해요.
- 커밋 메시지에 `Co-Authored-By: Claude`, `Generated with Claude Code` 같은 AI 관련 트레일러, 서명, 푸터를 절대 추가하지 않아요. 사용자가 명시적으로 요청한 경우에만 예외로 해요.
